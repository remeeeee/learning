Time Schedule

Time schedule is used to control the clients' on-line time.

"Account -> Time Schedule"

Time Schedule Name is used to define different time schedule rules.
Sunday to Saturday: This is used to set different on-line time schedules for business days. You can edit time schedule directly in the edit box or click the button besides edit box. In the popup "Time Table" dialog box, there are 24 time range choices. Note:"01:00" means Internet access is allowed from 00:00 to 01:00. "Apply to," means the time control of the day is only effective for checked choices.

Note: This step is only to define a time schedule rule. If you want to apply it to the specific account, you should edit the account's property and select the time schedule rule for this account.
