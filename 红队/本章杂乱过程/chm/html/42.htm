Cache Page

CCProxy's web caching is based on the IE cache storage technology. CCProxy will load and store the web pages into the IE cache files on the server.

Cache Update Time: To set the update interval of the cache.
Always Load from Cache: If this is checked, the HTTP proxy will always load web pages from the IE cache ignoring the cache update time.
Check the "IE" button if you wish to change the web cache path and size.
