Networks Page

Enable SOCKS4: To enable the socks4 and socks4a proxy service.
Disable External Users: To disable the users' access out of the LAN.
Server Bind IP Address: If the server is multiple-hosted, you can use a single IP address. "0.0.0.0" will select a server IP address automatically.
Bandwidth Control Apply for: To select which direction access will be limited by bandwidth control in the account management.
Web Sites for On-line Checking: To set the web sites that are used to check the Internet connection status.
Socket Idle Disconnect Minutes: To set the socket idle time out value.
Enable HTTP_X_FORWARDED_FOR: Add X-Forwarded-For in HTTP header.
